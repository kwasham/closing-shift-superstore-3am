# Store page brief — Sprint 4

## Goal
Ship a truthful, readable store-page package for the current launch-candidate build of **Closing Shift: Superstore 3AM**.

This brief is locked to:
- one supermarket
- 1–6 players
- short 8–10 minute rounds
- real disruption set only: blackout, false tasks, security alarms
- existing payoff loop: `Saved Cash`, XP, cosmetic nameplates, cosmetic lanyards

## Recommended title
**Closing Shift: Superstore 3AM**

## Recommended tagline
**Close the store. Survive the shift.**

## Short description
**1–6 players close a haunted supermarket at 3AM: finish tasks, survive blackout, false-task, and security-alarm disruptions, then cash out for cosmetics.**

## Long description
Clock in with 1–6 players and close a fluorescent supermarket at 3AM. Restock shelves, clean spills, take out trash, return carts, check the freezer, and save the register for last before the timer runs out.

The store does not stay normal for long. Blackouts cut visibility, false tasks waste time, and security alarms force the team to react fast while the shift keeps moving.

Finish shifts to earn **Saved Cash** and XP, then spend **Cash** on cosmetic nameplates and lanyards. Built for short, replayable 8–10 minute rounds with co-op panic, readable objectives, and late-night retail dread.

## Genre and mood recommendation
- Primary: **Co-op horror job sim**
- Secondary: **Short-session task horror**
- Mood: **fluorescent dread, late-night retail tension, social panic, false leads**

## Content-maturity note
Recommended disclosure:
- mild horror tension
- darkness / blackout states
- sudden alarms
- paranormal fake-out moments
- no graphic gore or explicit violence focus

## Positioning pillars
1. **Readable fantasy first**
   - players immediately understand they are closing a supermarket at night
2. **Task pressure plus disruption**
   - the tension comes from finishing work while the store turns hostile
3. **Short replay loop**
   - finish a shift, earn payout, unlock cosmetics, run it again
4. **Social confusion over combat**
   - the game sells panic, wrong calls, and last-minute teamwork, not fighting

## Icon brief
Create one clean icon that reads at a glance.
- Show checkout lane or aisle depth first.
- Keep the environment unmistakably retail.
- Use harsh fluorescent light with darkness pushing in at the edges.
- Optional worker silhouette is fine if it helps scale.
- Do not use monster close-ups, weapons, attack poses, or extra-location teases.

## Thumbnail briefs

### 1) Work shot — "Get the store closed"
Purpose: instantly explain the core loop.
- Show 2–3 players handling obvious tasks in the supermarket.
- Good props: stock boxes, carts, spill cleanup, freezer door, register lane.
- Keep the timer/task-pressure feeling readable.
- Suggested caption if needed: **Finish the task list**

### 2) Disruption shot — "The shift turns wrong"
Purpose: sell the horror hook honestly.
- Set the shot inside the same store during blackout or false-task confusion.
- Use darkness, wrong objective glow, or sudden alarm pressure.
- The supermarket layout must still read clearly.
- Suggested caption if needed: **Watch for false tasks**

### 3) Payoff shot — "Cash out and customize"
Purpose: show replay value without overpromising scope.
- Show results readability, payout landing, or visible nameplate/lanyard cosmetics.
- Keep the composition simple enough to read on mobile.
- Do not imply a large inventory or stat-build system.
- Suggested caption if needed: **Earn Cash for cosmetics**

## Screenshot guidance
