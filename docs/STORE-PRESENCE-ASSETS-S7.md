# Sprint 7 Store Presence Assets

## Purpose
Refresh the game's public-facing images so the Roblox surface finally matches the accepted visual direction inside the game.

## Asset pack
1. **Experience icon**
   - square composition
   - strong checkout / store-brand / threat read
   - readable even at small size

2. **Thumbnail A — Store at 3AM**
   - wide recognizable supermarket frame
   - strong signage and visual identity
   - no misleading one-off set dressing

3. **Thumbnail B — Blackout**
   - same store identity under emergency lighting
   - readable silhouette and contrast
   - mood without losing legibility

4. **Thumbnail C — Mimic tension**
   - local uncanny wrongness near a real taskable zone
   - should still feel like this game, not generic horror art

5. **Update / social shot**
   - before / after or "visual upgrade" style image
   - useful for release notes, social posts, and progress sharing

## Composition rules
- capture from the real shipped build whenever possible
- use spaces that now exist in the actual experience
- avoid shots that would disappoint players because they cannot find that scene in the game
- signage, palette, and lighting should match the accepted Sprint 6 / Sprint 7 art language

## Technical direction
- icon should be authored for the Roblox experience icon surface
- thumbnail images should follow Roblox's 16:9 format and target high-resolution export
- leave safe composition room for title treatment or UI overlays if used later

## Capture checklist
- one icon candidate set
- at least three final thumbnail candidates
- one update / social shot set
- one mobile-safe crop review
- record the in-game zone / camera setup used for each shot

## Approval rule
Do not approve the store-presence pack if it oversells fidelity the live game does not actually deliver.
