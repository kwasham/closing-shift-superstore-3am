# Sprint 6 Environment Kit and Vertical Slice Scope

## Critical-path polish order
1. lobby / entrance
2. checkout zone
3. one hero aisle
4. freezer section
5. stockroom corner

## Why this order
These are the spaces most likely to shape first impression, objective readability, and thumbnail capture value.

## Modular kit families
### 1. Architectural shell
- floor tile variants
- wall panel variants
- ceiling grid + fluorescent fixture module
- backroom door / employee-only door
- freezer doorway frame
- trim, corner, and baseboard pieces

### 2. Retail fixtures
- single shelf bay
- double shelf bay
- endcap shelf
- checkout counter
- bagging stand
- freezer / cooler case
- basket rack
- cart corral
- pallet / stock dolly

### 3. Small prop families
- boxes, tape, labels
- bottles, cartons, cans, bags
- baskets, carts, mop bucket, wet-floor sign
- receipt rolls, scanner accents, register clutter
- employee notice posters and pricing tags

### 4. Signage + decals
- aisle numbers
- product-category headers
- sale cards
- employee-only signage
- hazard stripes
- cooler labels / freezer warnings
- payroll / policy posters for back-of-house flavor

## Slice-zone requirements
### Lobby / entrance
- must establish brand, tone, and rule clarity immediately
- replace placeholder walls / floors / desk surfaces in the first player view
- include one strong branding element players remember

### Checkout zone
- must feel like the operational heart of the store
- the close-register objective needs strong visual importance
- queue rails, scanner accents, bags, signage, and terminal framing should help the zone feel authored

### Hero aisle
- must showcase the shelf kit and signage system
- should include at least one readable category identity
- avoid cluttering the lane so movement stays clean

### Freezer section
- must feel visually distinct via material, color, and lighting
- use colder light and reflective / glass cues

### Stockroom corner
- must communicate utility, staff access, and slight unease
- should bridge the normal store and horror tone

## Replacement rules
- Any highly visible primitive that survives in the polished slice must have a reason
- Preserve collision, interaction range, and task readability when replacing surfaces or props
- Do not move task-critical objects without matching engineering hooks

## Fallback rule
If asset production time gets tight, prioritize:
1. materials
2. silhouette improvements
3. signage
4. prop density
5. optional bespoke details

## Naming / organization guidance
- keep kit pieces grouped by family
- document any imported asset IDs
- separate decorative props from gameplay-critical nodes
- use clear names that content, engineer, and QA can all recognize
