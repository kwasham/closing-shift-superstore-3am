# Sprint 7 Full-Store Art Rollout

## Rollout intent
Take the accepted Sprint 6 style and apply it across all player-visible, player-traversed spaces so the experience no longer feels like a polished demo stitched onto a graybox shell.

## Priority tiers
### Tier A — must look production-ready this sprint
- lobby / entrance
- all main playable aisles
- checkout zone and immediate queue space
- freezer / cooler path
- stockroom corner and entered back-of-house space
- all transitions between those zones

### Tier B — must be visually consistent, even if lighter detail
- secondary aisle edges
- sightline background walls and ceiling runs
- visible staff access hall segments
- visible corners adjacent to active task nodes

### Tier C — can stay simpler if hidden or non-traversed
- unseen backsides of fixtures
- sealed-off or never-seen support spaces
- distant background views outside the playable frame

## Rollout order
1. remove the most obvious visible primitives from Tier A
2. extend floor / wall / ceiling language across the main store
3. apply signage and wayfinding pass
4. dress remaining Tier A and Tier B with props and decals
5. run a continuity pass for transitions, sightlines, and capture angles
6. only then spend time on optional micro-props

## Replacement matrix
### Architecture
- placeholder floors -> commercial tile / sealed utility floor by zone
- plain walls -> paneled / painted commercial wall treatment
- raw ceilings -> grid + fixture rhythm
- generic doors -> staff / freezer / service-door variants

### Fixtures
- primitive shelves -> approved modular shelf families
- primitive checkout blocks -> counter / bagging / scanner kit
- primitive freezer blocks -> cooler / glass-door family
- placeholder crates -> labeled box / pallet / dolly family

### Readability props
- aisle markers
- category signs
- checkout numbers
- sale / price cards
- wet-floor / hazard cues
- employee-only signage
- policy / payroll / notice posters for flavor

## Wayfinding goals
- players should be able to identify checkout, freezer, stockroom, and aisle structure at a glance
- signage should support memory during blackout recovery
- category cues should make the space feel like a store, not a generic corridor maze

## Preservation rules
- never bury task-critical nodes in clutter
- preserve path width and collision predictability
- do not move gameplay-critical objects without matching engineering support
- keep decorative dressing visually separate from task interaction affordances

## Fallback order if time compresses
1. materials / silhouette cleanup
2. signage / wayfinding
3. lighting continuity
4. medium prop density
5. high-detail clutter

## Proof expectation
Capture one before / after pair for every Tier A zone and at least one continuity shot showing that the full store now reads as one product.
