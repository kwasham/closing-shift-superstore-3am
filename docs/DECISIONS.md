# Decisions

## Decision log
- Use a text-first Rojo workflow instead of a Studio-only binary workflow.
- Keep MVP to one store map and one currency.
- Server owns round state, rewards, and dangerous events.
- Cosmetics come before strong power monetization.
- OpenClaw uses `main` as orchestrator plus worker agents for design, engineering, content, and QA.
- 2026-04-05 — Sprint 7 style sentence locked: Closing Shift should now read as a fluorescent suburban supermarket that is modular, phone-legible, and subtly wrong—an ordinary late-night retail box where every player-traversed space shares the same signage grammar, material language, and lighting rhythm, and where the horror lands through contrast and disruption instead of bespoke hero clutter.
- 2026-04-05 — Sprint 7 scope is locked to full-store visual rollout and store presence only; no new gameplay systems, event mechanics, economy changes, or map-expansion creep are allowed in this sprint.
- 2026-04-05 — Sprint 7 rollout tiers are locked:
  - Tier A: lobby / entrance, all main playable aisles, checkout + queue, freezer / cooler path, stockroom corner / entered back-of-house space, and all transitions between them.
  - Tier B: secondary aisle edges, sightline background walls and ceiling runs, visible staff-hall segments, and visible corners adjacent to active task nodes.
  - Tier C: unseen fixture backsides, sealed support rooms, and distant background views outside the playable frame.
- 2026-04-05 — Primitive removal policy locked: visible Tier A primitives and normal Tier B sightline placeholders must be replaced this sprint; remnants are acceptable only when hidden from normal play, QA proof, and public capture.
- 2026-04-05 — Store-wide signage grammar locked around five repeated systems only: aisle markers, category headers, checkout numbers, staff / hazard signs, and one sale-card family.
- 2026-04-05 — Transition rules are locked so lobby, main floor, freezer, and stockroom read as one store with different operational zones, not separate art styles.
- 2026-04-05 — Sprint 7 public-facing assets must be sourced from real completed in-game angles; the icon, thumbnails, and update/social shots cannot oversell fidelity the live build does not deliver.
- 2026-04-05 — Performance rule locked: modular reuse beats bespoke hero art, and any art uplift that damages phone readability, traversal clarity, blackout readability, or mimic readability fails Sprint 7.
- 2026-04-05 — If Sprint 7 compresses, the fallback order is: materials / silhouette cleanup first, signage second, lighting continuity third, medium prop density fourth, optional micro-clutter last.
