# Content Worker — Sprint 6 Environment + UI Art Pass

## Mission
Turn the locked style direction into a visible, player-facing art lift that gets the game off primitives in the critical path.

## You own
1. modular environment dressing,
2. material and decal pass,
3. signage and category-language pass,
4. small-prop dressing,
5. HUD visual assets / iconography support,
6. hero-shot composition guidance.

## Required outputs
- polish pass for:
  - lobby / entrance
  - checkout zone
  - one hero aisle
  - freezer section
  - stockroom corner
- reusable shelf / counter / cooler / signage kit decisions
- decal and sign pack
- small-prop pass that improves believability without hurting readability
- UI visual treatment assets or documented style hooks needed by engineering
- hero-shot / thumbnail composition notes

## Constraints
- Do not break interaction ranges or hide objective readability
- Do not turn Sprint 6 into a full-map beautification sprint
- Prefer reusable kits over one-off bespoke set pieces
- If using external or imported assets, document every dependency clearly
- Preserve the art-direction lock; do not drift into realism or unrelated horror styles

## Files to update
- `project/docs/ART-DIRECTION.md`
- `project/docs/HANDOFF-CONTENT.md`
- `project/docs/BACKLOG.md`
- `project/docs/RUNTIME-EVIDENCE.md`

## Proof expectations
Provide before / after captures or clear placement evidence for each polished zone.

## Definition of done
A first-time player entering the main slice should immediately feel that the game has a deliberate visual identity rather than a generic blockout.
